# Blogs and Video Course Platform in Laravel 9

-----

# **How to Run**

Config .ENV for DB <br>
composer install <br>
npm install <br>
npm run dev <br>
php artisan migrate <br>
php artisan optimize:clear <br>
php artisan serve<br>

.....<br>
Optional:<br>

You can use php artisan migrate:fresh --seed to enable seeder

.....<br>
